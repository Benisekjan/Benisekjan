Co je css
  • CSS (Cascading Style Sheets)- česky kaskádové styly
  • CSS šetří čas tím že můžeme použít stejné styly na více stránkách najednou
  • CSS udává jak bude vypadat celková stránka
Příklady

body {

  background-color: lightblue;

}

 

h1 {

  color: white;

  text-align: center;

}

 

p {

  font-family: verdana;

  font-size: 20px;

}
